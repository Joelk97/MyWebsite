Self driving car AI

The firt project I want to present is a little artificial intelligence, that I created during the JavaScript trainig. The simulation and the AI are made by following a tutorial of freeCodeCamp.org. The user-interface and the background elements are made by myself. I choose this project because I'm interested in reinforcement learning and I wanted to learn some JS basics, like OOP, Functions etc. by doing a funny project.
